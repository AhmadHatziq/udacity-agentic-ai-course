# udacity-agentic-ai-course
Project files used for the Agentic AI Nanodegree from Udacity 

There are 4 projects for this nanodegree: 
1. Trip Planner Agent 
2. Agentic Workflow for Project Management
3. Video Game Agent with access to RAG tool, uses state machines
4. Agentic Workflow for a Paper Company, uses cosine similarity to find similar sales quotes
